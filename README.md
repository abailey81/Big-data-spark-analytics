# Big Data Analytics with PySpark

End-to-end data engineering and analytics in Apache Spark: batch processing, graph analytics, and structured streaming.
Built for clarity, reproducibility, and hiring-manager review.

## Highlights
- Batch analytics (NYC taxi): cleaning, joins, aggregations, KPI tables.
- Graph analytics (zone network): shortest paths & PageRank (GraphFrames).
- Streaming analytics (NASA logs): windowing, watermarks, trigger intervals.
